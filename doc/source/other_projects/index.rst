================
 Other projects
================

The following are closely related projects that have heavily influenced the
design of datarray.  Both Larry and Pandas target slightly higher level
problems than datarray,  but the intended outcome is for datarray to provide
a base object on which projects like these can more easily build their
domain-specific tools with a common foundation.
